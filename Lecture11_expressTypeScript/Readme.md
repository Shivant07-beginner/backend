# 📦 Installations

> Whenever you create a new project, install these packages as needed.

## 🟢 Required

```bash
npm init -y
```

```bash
npm install express
```

## 🟡 Development Dependencies

```bash

```

```bash

```

```bash
npm install -D ts-node
```

```bash
npm install -D @types/node
```

```bash
npm install -D @types/express
```
```bash
 npm install -D tsx
```

## 🟠 Initialize TypeScript

```bash
npx tsc --init
```

---

# ▶ Running the Project

### JavaScript

```bash
node server.js
```

or

```bash
npx nodemon server.js
```

---

### TypeScript

```bash
npx ts-node src/server.ts
```

or

```bash
npx nodemon src/server.ts
```

# 📌 Commands Cheat Sheet

| Command | Purpose |
|----------|---------|
| `npm init -y` | Create package.json |
| `npm install express` | Install Express |
| `npm install -D nodemon` | Auto restart server |
| `npm install -D typescript` | Install TypeScript |
| `npm install -D ts-node` | Run TS directly |
| `npm install -D @types/node` | Node type definitions |
| `npm install -D @types/express` | Express type definitions |
| `npx tsc --init` | Generate tsconfig.json |
